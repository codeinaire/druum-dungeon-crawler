   - A feature flag name is wrong (e.g. `bevy_text` was renamed in 0.18). Fix: remove offending entry; consult the live manifest via the curl in plan Step 1.
   - A required feature (`bevy_winit`, `bevy_pbr`, or `bevy_asset`) is NOT pulled in transitively by `3d`. Fix: add explicit feature; the `Cargo.toml` already has a comment noting this possibility.
   - MSRV `1.85` is below Bevy 0.18.1's actual MSRV. Fix: bump `rust-toolchain.toml` channel and `Cargo.toml` `rust-version`.
